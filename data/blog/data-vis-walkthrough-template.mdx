Import Dependencies 🐍

Most projects rely on pre-written code libraries/modules whether written by the Notebook
author or by a third-party. When importing from libraries, it is common to use aliases
such as pd is used below as a shorthand for pandas.

```python
# Dependency Imports
import pandas as pd
import matplotlib.pyplot as plt
import json
from js import fetch
```

Helper function(s) 🐍

Helper function enable us to write clean human readable code.

```python
# Helper function which uses the browser's built-in fetch API
async def get_json(url):
    resp = await fetch(url)
    txt = await resp.text()
    return json.loads(txt)
```

## Helper function(s) 
this code requests data from a url with the (get_json(url)) .it does
this by (resp = await fetch(url)) sending a HTTP get request
and waiting for a response. when it gets a response (txt = await resp.text())
it uses this to get readable text which can then be displayed.
```python
# Helper function which uses the browser's built-in fetch API
async def get_json(url):
    resp = await fetch(url)
    txt = await resp.text()
    return json.loads(txt)
```

## Parse the tabulation into a tidy DataFrame
this code first creates a dataframe that organizes the column titels and the rows data. 
then diffines the function clean_value which replaces all of the nonsesns data with NA 
and converts real cencus data in to numbers. then it calls the function it just diffined 
and skips the rows and coloms that are the titel rows 
```python
df = pd.DataFrame(data[1:], columns=data[0])

def clean_value(x):
    if x in [None, "", "(X)", "*****"]:
        return pd.NA
    return pd.to_numeric(x, errors="coerce")

for col in df.columns:
    if col not in ["GEO_ID", "NAME", "us"]:
        df[col] = df[col].map(clean_value)
```

## Label maps
this code is a dictionary of translates.it translates the none human readable codes
to human readable names the names translate to: college/graduate school completion for people 25+,
bachelor's degree or higher for people 25+, disability prevalence for population 5+, 
residential stability (same house one year ago), naturalized citizenship among foreign-born, 
poverty rates, and population in households versus group quarters.
```python
labels = {
    "DP02_0058PE": "College or graduate school",
    "DP02_0068PE": "Bachelor's degree or higher",
    "DP02_0072PE": "With a disability",
    "DP02_0080PE": "Same house",
    "DP02_0093PE": "Naturalized citizen",
    "DP02_0125PE": "Below poverty level",
    "DP02_0154PE": "Population in households"
}
```

## Sort by total
first it creates an empty list plot_df. after that it goes throught every veriable in the translation dictionary 
and checks if it exsists in the dataframe. if it does then it ads the dictionary translation to it.then it converts 
back to a data frame and finaly sorts it by percent in decending order.
```python
plot_df = []
for var, label in labels.items():
    if var in df.columns:
        val = df.loc[0, var]
        if pd.notna(val):
            plot_df.append({"Measure": label, "Percent": float(val)})

plot_df = pd.DataFrame(plot_df).sort_values("Percent", ascending=False)
```

## Prepare for Data Visualization
this code creates a horisontal bar chart that shows the percentages in each topic. it creates the title the grid it 
is on on with the x and y lables.
```python
plt.style.use("seaborn-v0_8-whitegrid")
fig, ax = plt.subplots(figsize=(10, 6))

bars = ax.barh(plot_df["Measure"], plot_df["Percent"], color="#4C78A8")

ax.set_title("Selected ACS 2020–2024 DP02 National Percentages (5-Year Estimates)")
ax.set_xlabel("Percent")
ax.set_ylabel("")
ax.set_xlim(0, plot_df["Percent"].max() * 1.15)

for bar, value in zip(bars, plot_df["Percent"]):
    ax.text(
        bar.get_width() + 0.5,
        bar.get_y() + bar.get_height() / 2,
        f"{value:.1f}%",
        va="center"
    )
```
